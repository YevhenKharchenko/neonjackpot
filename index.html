CORE GAME IDEA

Drop Away is basically about fitting colored blocks in tight spaces without messing up.

  • You drag furniture-shaped blocks into a limited area. Each block has a color. Each stick figure is waiting for the correct color.

  • If you block the wrong path — you’re stuck. If you think ahead — everything flows.

  • There are no timers.
    No reflex challenges.
    It's all about figuring stuff out.

THE DROP MECHANIC

Look, what really matters in this game is:

  • What you drop first decides everything else

  • You don't have much room to work with, and it'll punish you for mistakes

  • One wrong drop can lock the entire board

  • Blocks interact with each other by position

  • Colors must reach the right stick figures

  • The challenge isn’t speed — it’s foresight

WHY IT’S ADDICTIVE

Players keep coming back to Drop Away because:

  • You always feel like there's gotta be a way to beat it

  • When you mess up, you can usually see what went wrong

  • When you finally solve it, you actually feel like you worked for it

  • The board “clicks” when solved correctly

It’s the kind of game where you restart immediately — not because you lost, but because you almost had it.
